# **App Name**: ResumeFlow

## Core Features:

- Welcome Page: Professionally designed welcome page with light/dark toggle, hero section, and clear 'Login' and 'Register' buttons.
- Authentication: Firebase-powered authentication pages (login and registration) using email/password and Google login.
- Job Seeker Dashboard: Dashboard to upload a resume (PDF) as a job seeker. View parsed resume information in a standardized and easily-readable manner.
- AI Job Matching Tool: AI to match uploaded resumes against existing job postings.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which translates to a vibrant blue (#3399FF).
- Background color: Light scheme using desaturated blue at HSL(210, 20%, 95%), equivalent to hex #F0F5FA.
- Accent color: Analogous hue around HSL(180, 80%, 40%), similar to a turquoise (#33CCCC), provides a contrast that complements the blue tones, without clashing.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- CSS files to mimic Tailwind style with utility classes, spacing, shadows, and rounded corners, promoting a clean and minimalist design. NO Tailwind.
- Use modern and minimalist icons to represent various job roles, industries, and functionalities.
- Subtle transitions and animations for page loading and UI interactions to improve user experience. It should  be written  with  React
